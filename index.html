Credit consumption by action type
Basic message / topic
~2 credits per interaction — simple Q&A or guided conversation step.
Generative answer
~10 credits — AI-generated response with or without document/web grounding.
Connector action
~5 credits per call — invoking SharePoint, a custom API, or Power Automate flow.
Autonomous trigger
~25 credits — unattended/scheduled runs. Always charged regardless of license type.
Complex exchange (example)
Generative answer + Graph grounding + 2 connector actions = ~22 credits for one exchange.
M365 Copilot users
Direct interactions in Teams/M365 apps = 0 credits. Autonomous runs still cost 25 credits.

Key capabilities
Agent inventory
Centralised view of all AI agents — Entra-registered agents, Agent Store agents, and shadow agents running without IT approval.
Agent identity
Each agent gets a unique Microsoft Entra ID, enabling precise access control to data and services — agents are managed exactly like users.
Security integration

Lifecycle governance
Full lifecycle control — deploy, update, suspend, or decommission agents with approval workflows and full change tracking.

How attach pricing works
Qualifying base licenses
A user holding Sales Enterprise, Customer Service Enterprise, Field Service, or Business Central Premium qualifies. Additional D365 apps cost $20/month instead of standalone.
Example saving
Sales Enterprise ($105) + Customer Service Enterprise ($95) standalone = $200/month. With attach: $105 base + $20 = $125/user/month — saving $75 per user.
Copilot included
All CE apps include embedded Copilot — email drafts, meeting summaries, lead scoring, and natural language pipeline queries. No extra charge.
Licence per user
Each user only needs the apps relevant to their role. A salesperson doesn't need Field Service — licence precisely to avoid overspend.
